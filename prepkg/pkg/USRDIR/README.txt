Place yours baseq2 folder here with following structure:

NOTES:
    USRDIR - this folder do not create a new one.
    filenames case sensitive

    USRDIR/baseq2/
    ├── music/     (optional) 
    │   ├── 02.ogg (optional)
    │   ...
    │   └── 11.ogg (optional)
    ├── save/      (optional, game will store save files here) 
    ├── config.cfg (optional, will be created by game)
    ├── pak0.pak
    ├── pak1.pak
    └── pak2.pak
